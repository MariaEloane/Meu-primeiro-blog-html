1. O que é HTML

HTML significa HyperText Markup Language. É a linguagem usada para criar páginas da internet. Ele organiza textos, imagens, links e tudo que aparece em um site ou blog.

2. Para que serve o HTML

O HTML serve para: Criar sites e blogs, organizar textos e imagens, criar links entre páginas, ajudar o navegador a entender o conteúdo. Sem HTML, não existe site.

3. Como o HTML funciona

O HTML usa tags, que dizem ao navegador o que cada parte do texto é.

Exemplo:

<p>Este é um parágrafo</p>

4. Estrutura básica de um documento HTML

Todo site começa assim:

<!DOCTYPE html>

<html lang="pt-br">

<head>

<meta charset="UTF-8">

<title>Meu Blog</title>

</head>

<body>

<h1>Bem-vindo ao meu blog</h1>

</body>

</html>

5. Principais tags do HTML Títulos

Títulos

<h1> até <h6>

Usados para títulos e subtítulos.

Parágrafo

<p>

Usado para textos.

Link

<a href="https://site.com">Link</a>

Imagem

<img src="imagem.jpg" alt="descrição">

Listas

<ol> lista ordenada

<ul> lista não ordenada

Divisão

<div>

Usado para organizar o conteúdo.

6. O que um blog precisa ter

Um blog simples precisa de:

  • Nome do blog

  • Menu

  • Posts

  • Textos

  • Imagens

  • Rodapé

7. Estrutura simples de um blog

<header>

<h1>Meu Blog<h1>

</header>

<nav>

<a href="#">Início</a>

<a href="#">Sobre</a>

<a href="#">Contato</a>

</nav>

<main>

<article>

<h2>Título do Post</h2>

<p>Conteúdo do post...</p>

</article>

</main>

<footer>

<small>© 2026 - Meu Blog</small>

</footer>

8. Boas práticas básicas

  • Usar tags corretas

  • Organizar o código

  • Usar alt nas imagens

  • Usar títulos em ordem

9. Próximos passos
